# L-System-Artwork-Generator

## How to Run
1. **Create the Conda environment**
```bash
   conda create --name iat460 python=3.11
```
2. **Activate conda environment**
```bash
   conda activate iat460
```
3. **Install packages**
```bash
    pip install -r requirements.txt
```
4. **Run main.py**
```bash
python main.py
```
